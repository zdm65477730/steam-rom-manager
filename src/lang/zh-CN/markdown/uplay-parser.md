# UPlay 解析器

This 解析器 imports games from `UPlay` (also known as `Ubisoft Connect`) so that 艺术作品 can be chosen for them and they can be added into Steam. If it doesn't work it is most likely because Ubisoft has altered the structure of their game database, 在这种情况下 please let the developers of SRM know and we will resolve the issue.

## Compatibility

This 解析器 is currently functional on `Windows` systems only.
